# Static Website Hosting using S3 Bucket on AWS

## Step 1: Login to AWS Console
1. Open the AWS Management Console  
2. Sign in with your credentials  
3. Search for "S3" in the services section  

---

## Step 2: Create S3 Bucket
1. Click "Create bucket"  
2. Enter a globally unique bucket name  
   - Example: my-static-website-123  
3. Select region (choose closest region)  
4. Uncheck "Block all public access"  
5. Acknowledge the warning  
6. Click "Create bucket"

## Step 3: Upload Website Files
1. Open your bucket  
2. Click "Upload"  
3. Add files (index.html, CSS, JS, images)  
4. Click "Upload"

## Step 4: Enable Static Website Hosting
1. Go to "Properties" tab  
2. Scroll to "Static website hosting"  
3. Click "Edit"  
4. Enable hosting  
5. Set:
   - Index document: index.html  
   - Error document: error.html (optional)  
6. Save changes

## Step 5: Set Bucket Policy (Public Access)
1. Go to "Permissions" tab  
2. Scroll to "Bucket Policy"  
3. Add the following policy (replace bucket-name):

   <img width="1919" height="984" alt="Screenshot 2026-04-23 225513" src="https://github.com/user-attachments/assets/c8448641-51c3-4e43-b4ce-44a58db48dfa" />


```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Sid": "PublicReadAccess",
      "Effect": "Allow",
      "Principal": "*",
      "Action": ["s3:GetObject"],
      "Resource": ["arn:aws:s3:::bucket-name/*"]
    }
  ]
}
